# ВИСНОВОК

У ході виконання курсового проекту було спроектовано, розроблено та оптимізовано веб-орієнтовану систему інтелектуального аналізу та коригування текстів **«AI Text Analyzer»**. 

Створений програмний продукт є сучасним інструментом, що поєднує класичні правила лінгвістичного аналізу та передові технології штучного інтелекту для комплексного опрацювання текстової інформації.

### Основні результати та технічні досягнення проекту:

1. **Архітектура системи:**
   Розроблено клієнт-серверну архітектуру на основі сучасного технологічного стеку: **React (JavaScript/JSX)** для побудови гнучкого та адаптивного інтерфейсу користувача (SPA) та **FastAPI (Python)** як високопродуктивного та асинхронного веб-інтерфейсу API на бекенді.

2. **Гібридний підхід до обробки тексту:**
   Впроваджено дворівневу систему перевірки:
   * **Правиловий аналіз (Rule-based):** інтеграція бібліотеки `LanguageTool` дозволила здійснювати точну локальну перевірку орфографії, граматики та пунктуації для української мови без надмірних витрат мережевих ресурсів.
   * **Нейромережевий аналіз (LLM-based):** завдяки інтеграції з **Google Gemini API** (модель `gemini-flash-latest`) реалізовано інтелектуальні функції: стилістичне покращення тексту, визначення тональності й складності за шкалою CEFR, автоматичну генерацію заголовків, резюмування (короткий опис) та виділення ключових слів.

3. **Оптимізація та продуктивність (Performance):**
   * Завдяки використанню асинхронного програмування та функції `asyncio.gather` на сервері вдалося паралелізувати запити до Gemini API (покращення стилю, аналіз метрик та створення заголовка), що скоротило загальний час очікування відповіді користувачем більш ніж на 60%.
   * На фронтенді застосовано оптимізаційні техніки, зокрема затримку виконання автозапитів (**Debounce**) при перекладі, що запобігає перевантаженню каналів зв'язку та API.

4. **Робота з документами та звітування:**
   Реалізовано модулі зчитування текстового вмісту з файлів популярних форматів (`.docx` та `.pdf`) безпосередньо на сервері. Також розроблено сервіси для експорту результатів роботи у структуровані звіти форматів **PDF (FPDF)** та **Word (python-docx)**.

5. **Стабільність та безпека (Robustness):**
   У процесі модернізації виправлено критичні помилки рендерингу документів, зациклення оновлення станів (Render Loops) та витоки пам'яті. Впроваджено жорсткі ліміти на обсяг вхідного тексту (5000 символів), що робить систему стійкою до перевантажень та економить кошти на використання хмарного API.

### Практичне значення:
Розроблена система «AI Text Analyzer» має високу практичну цінність для студентів, копірайтерів, редакторів та науковців. Вона пропонує єдине інтегроване середовище для швидкої перевірки помилок, глибокого стилістичного аналізу та перекладу текстів з можливістю автоматичного формування друкованих звітів. 

Проект повністю відповідає поставленим вимогам, демонструє працездатність та готовність до практичного впровадження.
